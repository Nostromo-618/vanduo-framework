Getting Started v1.1.1

CDN (Fastest)

Load directly from jsDelivr — no download required:

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Nostromo-618/vanduo-framework@main/dist/vanduo.min.css">

<!-- JS -->
<script src="https://cdn.jsdelivr.net/gh/Nostromo-618/vanduo-framework@main/dist/vanduo.min.js"></script>
<script>Vanduo.init();</script>

Tip: Pin to a version for production:
@v1.1.1 instead of @main

Download

Download the dist/ folder and include locally:

<!-- CSS -->
<link rel="stylesheet" href="dist/vanduo.min.css">

<!-- JS -->
<script src="dist/vanduo.min.js"></script>
<script>Vanduo.init();</script>

The dist/ folder is self-contained with CSS, JS, fonts, and icons.

Source Files

For development or modular imports:

<!-- Full framework -->
<link rel="stylesheet" href="css/vanduo.css">
<script src="js/vanduo.js"></script>
<script>Vanduo.init();</script>

<!-- Or pick individual modules -->
<link rel="stylesheet" href="css/core/colors.css">
<link rel="stylesheet" href="css/components/buttons.css">

Theme Switching

Vanduo supports light, dark, and system themes. Preference is stored in localStorage. 

<!-- Select-based toggle -->
<select data-toggle="theme">
  <option value="system">System</option>
  <option value="light">Light</option>
  <option value="dark">Dark</option>
</select>

<!-- Or button toggle (cycles modes) -->
<button data-toggle="theme">Toggle</button>

How it works:

  • data-toggle="theme" binds the element
  • Preference saved to vanduo-theme-preference key
  • Sets [data-theme="dark"] on <html>
  • System mode removes the attribute, deferring to prefers-color-scheme

Data Attributes Reference

JavaScript components are configured via data-* attributes:

Attribute Component
data-modal="#id" Modal trigger
data-dismiss="modal" Modal close button
data-tooltip="text" Tooltip content
data-tooltip-placement="top|right|bottom|left" Tooltip position
data-toggle="theme" Theme switcher
data-toggle="font" Font switcher
data-collapsible Collapsible container
data-speed="0.5" Parallax layer speed

JavaScript API

All components register with the global Vanduo object:

// Re-initialize a component after DOM changes
Vanduo.reinit('dropdown');

// Destroy all component instances (cleanup)
Vanduo.destroyAll();

// Register a custom component
Vanduo.register('myComponent', {
  init: function() { /* ... */ }
});

// Show a toast notification
Vanduo.components.toast.show({
  title: 'Success',
  message: 'Item saved.',
  type: 'success'
});

CSS State Classes:

  • .is-open — open state (modals, dropdowns, sidenav)
  • .is-active — active state (tabs, accordion items)
  • .disabled — disabled state

Icons

Vanduo includes Phosphor Icons — a flexible icon family with 1,500+ icons in 6 weights. Icons are bundled locally with no external dependencies.

Quick Start

Include the icons stylesheet and use <i> elements with the appropriate classes:

<!-- Include default weights (regular + fill) -->
<link rel="stylesheet" href="css/icons/icons.css">

<!-- Use icons -->
<i class="ph ph-heart"></i>
<i class="ph-fill ph-star"></i>
<i class="ph ph-house"></i>

Import Options

<!-- Default (regular + fill) - Recommended -->
<link rel="stylesheet" href="css/icons/icons.css">

<!-- All 6 weights (~3MB) -->

<!-- Individual weights -->

CSS Variable Usage

/* Direct scale access */
.my-element {
  background: var(--primary-0);  /* Lightest */
  color: var(--primary-9);       /* Darkest */
}

/* Semantic aliases (recommended) */
.my-button {
  background: var(--color-primary);
  border-color: var(--color-primary-dark);
}

/* Hover states using scale */
.my-button:hover {
  background: var(--primary-6);
}

.my-button:active {
  background: var(--primary-7);
}

Switching Fonts

Use the data-font attribute on the HTML element:

<!-- Set font on HTML element -->
<html data-font="ubuntu">

<!-- Or use the Theme Customizer (paint roller icon) to preview and select fonts -->
<button data-theme-customizer-trigger>
    <i class="ph ph-paint-roller"></i>
</button>

Font preference is persisted to localStorage and restored on page load.

Adding More Google Fonts

Need more fonts? You have two options:

Option 1: Self-Host (Recommended)

Download fonts from Google Fonts and add them to your fonts/ folder. Follow the same pattern as existing fonts: 

1. Create folder: fonts/my-font/
2. Add woff2 files
3. Add @font-face rules to css/core/fonts.css
4. Add [data-font="my-font"] selector to css/core/typography.css
5. Register in js/components/font-switcher.js
Option 2: Google Fonts CDN

For quick prototyping, use Google Fonts CDN link in your HTML head:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;500;700&display=swap" rel="stylesheet">

<style>
  [data-font="roboto"] {
    --font-family-sans: 'Roboto', sans-serif;
  }
</style>

Theme Customizer

The Theme Customizer is a powerful component that allows users to personalize the framework's appearance in real-time. It provides controls for primary color, neutral color, border radius, font family, and color mode (light/dark/system).

Quick Start

Add a trigger button anywhere in your page to open the customizer panel:

<!-- Trigger button -->
<button class="vd-theme-customizer-trigger" 
        data-theme-customizer-trigger 
        aria-label="Open theme customizer">
    <i class="ph ph-paint-roller"></i>
</button>

<!-- Include the JS component -->
<script src="js/components/theme-customizer.js"></script>

Try it: Click the icon in the navbar to open the Theme Customizer.

Features

  • Primary Color: 18 color options from the Open Color palette
  • Neutral Color: 5 gray scale variants (Gray, Slate, Zinc, Neutral, Stone)
  • Border Radius: 5 presets (0, 0.125rem, 0.25rem, 0.375rem, 0.5rem)
  • Font Family: 10 bundled fonts including JetBrains Mono (default) plus Google Fonts collection
  • Color Mode: System (default), Dark, or Light

All preferences are persisted to localStorage and restored on page load.

Available Primary Colors

Black Red Orange Amber Yellow Lime Green Emerald Teal Cyan Sky Blue Indigo Violet Purple Fuchsia Pink Rose

Available Neutral Colors

Gray (default) Slate Zinc Neutral Stone

Neutral colors affect backgrounds, borders, and text throughout the framework.

localStorage Keys

Key Default Description
vanduo-primary-color cyan Primary brand color
vanduo-neutral-color gray Neutral/gray scale
vanduo-radius 0.25 Border radius scale
vanduo-font-preference jetbrains-mono Font family
vanduo-theme-preference system Color mode

JavaScript API

// Access the component
const customizer = Vanduo.components.themeCustomizer;

// Open/close programmatically
customizer.open();
customizer.close();
customizer.toggle();

// Get current state
const state = customizer.getState();
// Returns: { primary, neutral, radius, font, theme }

// Set values programmatically
customizer.setPrimary('violet');
customizer.setNeutral('slate');
customizer.setRadius('0.375');
customizer.setFont('inter');
customizer.setTheme('dark');

// Reset to defaults
customizer.reset();

Data Attributes

The Theme Customizer sets data attributes on the <html> element:

Attribute Example Values Description
data-primary cyan, violet, rose Remaps --primary-* CSS variables to the selected color
data-neutral gray, slate, zinc Remaps --gray-* CSS variables to the selected neutral
data-radius 0, 0.125, 0.25, 0.375, 0.5 Sets --radius-scale for component border-radius
data-font jetbrains-mono, inter, fira-sans, ubuntu, open-sans, rubik, titillium-web Sets --font-family-base
data-theme light, dark, (removed for system) Forces light/dark mode or defers to prefers-color-scheme

Events

Listen for customizer events:

// Panel opened
document.addEventListener('themeCustomizer:open', () => {
    console.log('Customizer opened');
});

// Panel closed
document.addEventListener('themeCustomizer:close', () => {
    console.log('Customizer closed');
});

// Any setting changed
document.addEventListener('themeCustomizer:change', (e) => {
    console.log('Changed:', e.detail);
    // { type: 'primary', value: 'violet' }
});

// Settings reset
document.addEventListener('themeCustomizer:reset', () => {
    console.log('Settings reset to defaults');
});

Search

                        
A configurable client-side search component with keyboard navigation,
                            debounced
                            input, category grouping, and full accessibility support. Perfect for documentation sites,
                            knowledge bases, or any content-heavy application.


                        

                            

                                

                                    

                                        
Quick Start

                                    

                                    

                                        
Include the CSS and JS files, then add the search markup:

                                        
<!-- CSS -->
<link rel="stylesheet" href="css/components/doc-search.css">

<!-- HTML -->
<div class="vd-doc-search">
    <div class="vd-doc-search-input-wrapper">
        <i class="ph ph-magnifying-glass vd-doc-search-icon"></i>
        <input type="search" class="vd-doc-search-input"
               placeholder="Search..."
               aria-label="Search"
               aria-controls="search-results"
               aria-expanded="false"
               autocomplete="off">
        <kbd class="vd-doc-search-shortcut">⌘K</kbd>
    </div>
    <div class="vd-doc-search-results" id="search-results"
         role="listbox" aria-label="Search results"></div>
</div>

<!-- JS -->
<script src="js/components/doc-search.js"></script>

                                        
The component auto-initializes on
                                            page load
                                            and indexes all <section> elements with id
                                            attributes.

                                    

                                

                            


                            

                                

                                    

                                        
Features

                                    

                                    

                                        
    
                                            
  • Keyboard Navigation: Arrow keys, Enter to select,
                                                Escape to close
    • 
                                            
  • Keyboard Shortcut: ⌘K (Mac) /
                                                Ctrl+K (Windows) to focus
                                            
    • 
                                            
  • Debounced Input: 150ms delay prevents excessive
                                                searches
    • 
                                            
  • Category Grouping: Results grouped by section with
                                                icons
    • 
                                            
  • Highlight Matching: Search terms highlighted in results
                                            
    • 
                                            
  • ARIA Accessible: Full screen reader support
    • 
                                            
  • Dark Mode: Automatic theme adaptation
    • 
                                            
  • Reduced Motion: Respects
                                                prefers-reduced-motion
                                            
    • 
                                        

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
Custom Instance

                                    

                                    

                                        
Create a custom search instance with your own data and callbacks:

                                        
// Create custom search instance
const mySearch = Search.create({
    // Custom selectors
    inputSelector: '.my-search-input',
    resultsSelector: '.my-search-results',
    
    // Custom data source (optional)
    data: [
        {
            title: 'Getting Started',
            content: 'Learn how to install...',
            category: 'Guides',
            url: '#getting-started'
        },
        // ... more items
    ],
    
    // Callbacks
    onSelect: (item) => {
        console.log('Selected:', item);
        window.location.hash = item.url;
    },
    onSearch: (query, results) => {
        console.log(`Found ${results.length} results`);
    },
    onOpen: () => console.log('Search opened'),
    onClose: () => console.log('Search closed')
});

                                    

                                

                            


                            

                                

                                    

                                        
Configuration Options

                                    

                                    

                                        

                                            

                                                
                                                    

                                                        Option
                                                        Default
                                                        Description
                                                    

                                                
                                                
                                                    

                                                        inputSelector
                                                        .doc-search-input
                                                        Search input element selector
                                                    

                                                    

                                                        resultsSelector
                                                        .doc-search-results
                                                        Results container selector
                                                    

                                                    

                                                        data
                                                        null
                                                        Custom data array (auto-indexes DOM if null)
                                                    

                                                    

                                                        debounceMs
                                                        150
                                                        Debounce delay in milliseconds
                                                    

                                                    

                                                        maxResults
                                                        20
                                                        Maximum results to display
                                                    

                                                    

                                                        minQueryLength
                                                        2
                                                        Minimum characters to trigger search
                                                    

                                                
                                            

                                        

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
JavaScript API

                                    

                                    

                                        
// Access the default instance
const search = DocSearch;

// Or create a custom instance
const search = Search.create({ /* options */ });

// Methods
search.open();           // Open/focus search
search.close();          // Close results
search.clear();          // Clear input and results
search.destroy();        // Remove all event listeners

// Update search index (after DOM changes)
search.reindex();

// Search programmatically
search.search('query');

// Get current state
const isOpen = search.isOpen;

Image Box

                        
A smooth "lightbox" implementation for enlarging images. Supports dual-image
                            mode with separate thumbnails and full-size images. When users click on an image, Image Box
                            centers it and smoothly scales it up. Dismiss by clicking, pressing ESC, or scrolling away.
                        


                        
                        

                            

                                

                                    

                                        
Gallery
                                            with Dual Images

                                        
Thumbnail → Full-size on click

                                    

                                    

                                        
This gallery demonstrates the dual-image feature. Small
                                            thumbnails are displayed, but clicking opens the full-resolution version.
                                            Includes landscape and portrait orientations.


                                        
                                        

                                            
                                            

                                            
                                            

                                            
                                            

                                            
                                            
                                        

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
Quick Start

                                    

                                    

                                        
Add the data-image-box attribute to any image you want to make
                                            enlargeable:

                                        



                                        
The component auto-initializes on
                                            page load.
                                            No JavaScript setup required.

                                    

                                

                            


                            

                                

                                    

                                        
Features

                                    

                                    

                                        
    
                                            
  • Dual Image Support: Use
                                                data-image-box-full-src for separate thumbnail and
                                                full-size images
                                            
    • 
                                            
  • Smooth Transitions: CSS-powered scale animation with
                                                cubic-bezier easing
    • 
                                            
  • Zoom Cursor: Magnifying glass icon on hover indicates
                                                enlargement
    • 
                                            
  • Multiple Close Methods: Click image, press ESC, or
                                                scroll away
    • 
                                            
  • Scroll Detection: Auto-closes when scrolling more than
                                                50px
    • 
                                            
  • Backdrop Blur: Beautiful frosted glass effect behind
                                                enlarged image
    • 
                                            
  • Captions: Optional captions with
                                                data-image-box-caption
                                            
    • 
                                            
  • Accessible: Full keyboard navigation and ARIA
                                                attributes
    • 
                                        

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
Gallery Grid

                                    

                                    

                                        
Create responsive image galleries using the
                                            .image-box-gallery class. The grid automatically adjusts
                                            columns based on available space.
                                        

                                        



  
  
  
  
  
  
  


                                        
The gallery uses CSS Grid with
                                            auto-fit
                                            columns (minimum 120px). Add data-orientation="portrait" for
                                            3:4 aspect ratio images.

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
JavaScript API

                                    

                                    

                                        
// Access the component
const imageBox = Vanduo.components.imageBox;

// Open an image programmatically
imageBox.open(document.querySelector('[data-image-box]'));

// Close the image box
imageBox.close();

// Re-initialize after adding new images to DOM
Vanduo.reinit('imageBox');

// Destroy and clean up
imageBox.destroy();

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
CSS Customization

                                    

                                    

                                        
Override CSS variables to customize the appearance:

                                        
:root {
  /* Backdrop styling */
  --image-box-backdrop-bg: rgba(0, 0, 0, 0.9);
  --image-box-backdrop-blur: 8px;
  
  /* Animation duration */
  --image-box-transition-duration: 0.3s;
  
  /* Max size of enlarged image (default: 50% of viewport) */
  --image-box-max-width: 50vw;
  --image-box-max-height: 50vh;
}

/* Gallery grid customization */
.image-box-gallery {
  gap: 21px; /* Increase spacing */
}

/* Active/selected thumbnail */
.image-box-gallery .image-box-trigger.is-active {
  outline: 3px solid var(--color-primary);
  transform: scale(1.05);
}

                                    

                                

                            

                        


                        

                            

                                

                                    

                                        
Data Attributes

                                    

                                    

                                        

                                            

                                                
                                                    

                                                        Attribute
                                                        Type
                                                        Description
                                                    

                                                
                                                
                                                    

                                                        data-image-box
                                                        Boolean
                                                        Enables Image Box functionality on the image
                                                    

                                                    

                                                        data-image-box-full-src
                                                        String
                                                        URL of full-size image to display in lightbox (optional,
                                                            uses src if not set)
                                                    

                                                    

                                                        data-image-box-src
                                                        String
                                                        Alternative source URL (fallback if full-src not set)
                                                    

                                                    

                                                        data-image-box-caption
                                                        String
                                                        Caption displayed below the enlarged image
                                                    

                                                    

                                                        data-orientation
                                                        String
                                                        Use "portrait" for 3:4 aspect ratio gallery thumbnails
