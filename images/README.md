# Vanduo Framework - Images Directory

## SEO Images

This directory contains placeholder images for SEO and social media sharing. Replace these with your actual brand assets.

### Required Images

| File | Size | Purpose |
|------|------|---------|
| `og-image.png` | 1200x630px | Open Graph image for Facebook, LinkedIn |
| `twitter-card.png` | 1200x600px | Twitter Card image |
| `logo.png` | 512x512px | Organization logo for structured data |

### Favicon Files

Favicon files should be placed in the root directory:

| File | Size | Purpose |
|------|------|---------|
| `/favicon.ico` | 16x16, 32x32, 48x48 | Legacy favicon |
| `/favicon-16x16.png` | 16x16px | Modern browsers |
| `/favicon-32x32.png` | 32x32px | Modern browsers |
| `/apple-touch-icon.png` | 180x180px | iOS home screen |

### Generating Favicons

You can generate favicons from a single source image using online tools:
- https://realfavicongenerator.net/
- https://favicon.io/

### Image Guidelines

**Open Graph Image (og-image.png)**
- Dimensions: 1200x630px
- Format: PNG or JPG
- File size: < 1MB
- Content: Framework name, tagline, visual branding

**Twitter Card Image (twitter-card.png)**
- Dimensions: 1200x600px (2:1 ratio)
- Format: PNG or JPG
- File size: < 1MB
- Content: Similar to OG image but optimized for Twitter's aspect ratio

**Logo (logo.png)**
- Dimensions: 512x512px (square)
- Format: PNG with transparency
- File size: < 100KB
- Content: Clean, recognizable brand mark

### Placeholder Notice

The current images are placeholders. Update them with your actual brand assets before deploying to production.
