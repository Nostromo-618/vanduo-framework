Essential just like water is.

Pure HTML, CSS, JS framework for designing beautiful static websites. No dependencies, no build tools, just clean and simple code.

Features

Zero Dependencies

Pure HTML, CSS, and JavaScript. No build tools, no package managers, no complexity.

Fibonacci Design

Spacing, typography, and grid layouts built on the Golden Ratio and Fibonacci sequence for natural harmony.

Modular CSS

Import the full framework or pick only the components you need. No bloat, just what you use.

Accessibility First

ARIA attributes, focus-visible states, reduced motion support, and semantic HTML built-in.

Dark Mode Ready

Light, dark, and system themes with smooth transitions. Includes a customizable theme switcher component.

Lightweight & Fast

Minimal footprint with optimized CSS. No JavaScript required for styling—scripts are optional enhancements.

Phosphor Icons

What's New in v1.1.0

CSS Namespace

All CSS classes now use the vd- prefix to prevent collisions with other libraries.

<!-- v1.0.x (old) -->
<button class="btn btn-primary">Click</button>

<!-- v1.1.0 (new) -->
<button class="vd-btn vd-btn-primary">Click</button>
View Migration Guide

ESM & CJS Modules

Three build formats now available: IIFE, ESM, and CommonJS for modern bundler compatibility.

<!-- IIFE (Classic) -->
<script src="vanduo.min.js"></script>

<!-- ESM -->
<script type="module">
  import { init } from './vanduo.esm.min.js';
  init();
</script>

Manual Initialization

Components no longer auto-initialize. You control when Vanduo starts.

<script src="vanduo.min.js"></script>
<script>
  // Initialize when ready
  Vanduo.init();
  
  // Cleanup when needed (SPAs)
  // Vanduo.destroyAll();
</script>

Design Tokens

Centralized CSS custom properties with --vd- prefix for consistent theming.

:root {
  --vd-color-primary: #22b8cf;
  --vd-color-primary-dark: #1098ad;
  --vd-spacing-8: 8px;
  --vd-shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
}
